# Meal Finder

consume API Website with webpack, bootstrap 5, javascript

## Cloning folder

Get the project with gitbash

```bash
  git clone https://github.com/fannr/meal-finder.git
```

## Deployment

Download all package in folder

```bash
  npm install
```

## Running Tests

To run tests, run the following command

```bash
  npm run start-dev
```
